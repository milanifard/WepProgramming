HTML Links

  • a: Use the a element to define a link
  • href:Use the href attribute to define the link address
  • target: Use the target attribute to define where to open the linked document
  • img:Use the img element (inside a) to use an image as a link
  • mailto: Use the mailto: scheme inside the href attribute to create a link that opens the user's email program

HTML Lists

  • Unordered HTML List: An unordered list starts with the ul tag. Each list item starts with the li tag.
  • Ordered HTML List An ordered list starts with the ol tag. Each list item starts with the li tag.
