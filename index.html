خواندن این مطلب ۱ دقیقه زمان می برد

در زبان HTML می‌توانیم با استفاده از تگ‌هایی خاص، متن را فرم‌دهی کنیم. از جمله مهمترین این تگ‌ها می‌توان به: 

<b> bold text 
<strong> Important text 
<i> Italic text 
<em> Emphasized text 
<mark> Marked text 
<small> Small text 
<del> Deleted text 
<ins> Inserted text

و … اشاره کرد.

کامنت گذاری در HTML

اگر تا کنون تجربه برنامه نویسی داشته‌اید، مطمئنا تا به حال برایتان پیش آمده است که به کدتان کامنت‌هایی را اضافه کنید و آن را مستندسازی کنید. خوشبختانه این امکان در HTML هم وجود دارد.
برای استفاده از قابلیت کامنت گذاری در HTML باید از تگ زیر استفاده کنید: 

<!- - - متن دلخواهتان - - - >

استفاده از رنگ‌ها در HTML

برای استفاده از رنگ‌ها دو قابلیت پیش رویتان قرار دارد. یکی از آنها استفاده از نام خود رنگ است و دیگری استفاده از یک عدد RGB و انتخاب سه رنگ برای میزان شدت و رنگ آن است و با ترکیب آنها رنگ دلخواهتان را بسازید.
برای استفاده از رنگ‌ها می‌توانید از مثال زیر استفاده کنید: 

<h1 style="background-color:DodgerBlue;">Hello World</h1>
<p style="background-color:Tomato;">Lorem ipsum...</p>

سپاسگزاریم که تا اینجای مطلب همراه من بودید. امید است برای شما مفید بوده باشد.

یک tooltip به پاراگراف زیر اضافه کنید که متن سلام را نشان دهد:
<p ="About Us">hello.</p>
یک مقدار برای لینک زیر بگذارید تا در تب جدید باز شود:
<a target="">Link to new tab</a>
آدرس تصویر را با چه ویژگی مشخص میکنیم؟
<img ="/images/sample.jpg" />
لیبل دکمه را با چه ویژگی مشخص میکنیم؟
<input type="submit" ="ثبت نام" />
