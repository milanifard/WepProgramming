HTML Images


اگر کاربر به دلایلی نتواند آن را مشاهده کند (به دلیل اتصال کند، خطا در ویژگی src یا اگر کاربر از صفحه‌خوان استفاده می‌کند)، ویژگی alt یک متن جایگزین برای یک تصویر ارائه می‌کند.

با نقشه های تصویری، می توانید مناطق قابل کلیک روی یک تصویر اضافه کنید. تگ map یک تصویر-نقشه را تعریف می کند. تصویر-نقشه تصویری با مناطق قابل کلیک است.

تقریباً روی هر عنصر HTML می توان یک تصویر پس زمینه را مشخص کرد. برای افزودن یک تصویر پس زمینه در HTML، از ویژگی CSS background-image استفاده کنید.

اگر تصویر پس زمینه کوچکتر از عنصر باشد، تصویر به صورت افقی و عمودی تکرار می شود تا زمانی که به انتهای عنصر برسد.

اگر می‌خواهید تصویر پس‌زمینه کل عنصر را پوشش دهد، می‌توانید ویژگی پس‌زمینه اندازه را برای پوشش تنظیم کنید. همچنین، برای اطمینان از اینکه کل عنصر همیشه پوشیده است، ویژگی backgroundattachment را روی fixed تنظیم کنید

HTML5 عنصر picture را برای افزودن انعطاف پذیری بیشتر هنگام تعیین منابع تصویر معرفی کرد. عنصر picture حاوی تعدادی عنصر source است که هر کدام به منابع تصویر متفاوتی اشاره دارند. به این ترتیب مرورگر می تواند تصویری را انتخاب کند که به بهترین شکل با نمای فعلی و یا دستگاه مطابقت دارد.

همیشه یک عنصر img را به عنوان آخرین عنصر فرزند عنصر picture مشخص کنید. عنصر img توسط مرورگرهایی استفاده می شود که از عنصر picture پشتیبانی نمی کنند، یا اگر هیچ یک از تگ های source مطابقت نداشته باشد.
