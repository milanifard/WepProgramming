قلمرو اعتبار متغیرها


متغیرها در جاوااسکریپت

Javascript تنها دو نوع متغیر گلوبال و درون function دارد.
  • وقتی متغیر درون بدنه‌ی script تعریف شود، متغیر global خواهد بود.
  • اگر متغیر درون function تعریف شود، فقط در اسکوپ function اعتبار خواهد داشت.

Hoisting

دستورات declaration متغیرها همواره در ابتدای اجرای برنامه، اجرا می‌شوند. به بیان دیگر، خطوط ابتدایی برنامه، میزبان دستورات مربوط به تعریف متغیر می‌شود.
تفاوت بین declaration و initialization متغیرها
  • پدیده‌ی hoisting فقط برای declaration رخ می‌دهد. اگر آن متغیر مقداردهی شده باشد، مقدار مورد نظر hoist نمیشود و متغیر undefined می‌ماند.

کلمه کلیدی let

  • اگر بخواهیم یک متغیر فقط درون یک block تعریف شود و بیرون از آن نامعتبر باشد، از کلمه کلیدی let استفاده میکنیم.
  • کلمه کلیدی let برای متغیرهای فانکشن و گلوبال تغییر خاصی ایجاد نمیکند.
  • درون یک اسکوپ، نمیتوان اول از let و بعد از var استفاده کرد. زیرا خطای redeclaration رخ میدهد.
