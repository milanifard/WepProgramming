Color/text-align/text-indent

  • The color property is used to set the color of the text.

  • The text-align property is used to set the horizontal alignment of a text (left, right, center, justify). When the text-align property is set to "justify", each line is stretched so that every line has equal width, and the left and right margins are straight

  • The text-indent property is used to specify the indentation of the first line of a text (ex: text-indent: 50px;).
