ساخت فرم درHTML

عنصر form HTML فرمی را تعریف می کند که برای جمع آوری کاربر استفاده می شود ورودی عناصر فرم انواع مختلفی از عناصر ورودی هستند، مانند فیلدهای متنی، چک باکس ها، دکمه های رادیویی، دکمه های ارسال و موارد دیگر.
عنصر input مهمترین عنصر فرم است. را عنصر input بسته به ویژگی type می تواند به روش های مختلفی نمایش داده شود مشخصه target مشخص می کند که آیا نتیجه ارسال شده در یک برگه مرورگر جدید، یک فریم یا باز می شود در پنجره فعلی را متد ویژگی متد HTTP (GET یا POST) را مشخص می کند که در هنگام استفاده از آن ارسال اطلاعات فرم:
