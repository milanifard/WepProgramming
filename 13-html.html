کار با input

در این آموزش با input ها آشنا میشویم. هرزمان که بخواهیم از کاربر ورودی دریافت کنیم از تگ input استفاده میکنیم. در استفاده از این تگ میتوان از کلاس های موجود در bootstrap نیز بهره برد. حال به شروع آموزش میپردازیم.

فرض کنید میخواهیم از کاربر نام کاربری و رمز عبور اش را دریافت کنیم. برای اینکار نیاز به یک فرم داریم؛ برای اینکار میتوان از تگ form استفاده نمود. قطعه کدی مانند کد زیر مینویسیم:

مشاهده میکنیم که خروجی بصورت زیر است:

در خصیصه type مقدار های متفاوتی میتوان گذاشت مانند عدد و رمز. این خصیصه همانطور که از اسمش مشخص است، برای مشخص کردن نوع ورودی بکار میرود. برای گذاشتن یک عنوان هم میتوان از تگ legend استفاده کرد.

حال با داشتن ورودی میتوان از طریق اسکریپت نویسی عملیات هایی انجام داد در این مورد. در انتها نیز به بررسی type های مختلف میپردازیم:
